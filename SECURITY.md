# Security Policy

We take the security of FirstMove seriously. If you believe you have found a security vulnerability, please report it to us responsibly as outlined below.

## Supported Versions

Only the latest release of FirstMove is currently actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v3.x    | :white_check_mark: |
| < v3.0  | :x:                |

## Reporting a Vulnerability

**Please do not report security vulnerabilities via public GitHub issues.**

Instead, please send all security reports to **akshayjadhav237237@gmail.com** with details about:
- The nature of the vulnerability.
- Steps to reproduce it (with code snippets or proof of concept if applicable).
- The potential impact.

We will acknowledge receipt of your report within 48 hours and work with you to resolve the issue as quickly as possible.
