# FirstMove — Multi-Agent Decision Intelligence

Get three independent AI perspectives on your startup idea before you build the wrong thing.

FirstMove harnesses multi-agent debate to surface critical assumptions and unknowns in your business idea. Instead of a single opinion, three AI agents evaluate your concept from different angles, exposing blind spots and highlighting what you should validate first.

## What It Does

- **Multi-Agent Debate**: Three AI agents independently analyze your startup idea
- **Diverse Perspectives**: Each agent brings a different lens (market, technical, execution)
- **Decision Intelligence**: Surface key assumptions and unknowns worth testing
- **Actionable Insights**: Get concrete next steps before investing in building

## Quick Start

### Prerequisites
- Node.js (v16 or higher)
- npm

### Installation

```bash
npm install
```

### Running the Development Server

```bash
npm run dev
```

The application will start at `http://localhost:5173` (or the next available port).
